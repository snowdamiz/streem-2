<objective>
Create the `streem` meta-package in `packages/streem/` — a single-import entry point that flat re-exports all developer-facing primitives from `/core`, `/dom`, and `/streams`. Includes three entry points for the JSX subpath resolution required by `jsxImportSource: "streem"`.

Purpose: Developers should not need to know which sub-package a primitive comes from. `import { signal, render, fromWebSocket } from 'streem'` must work.
Output: `packages/streem/` package with `dist/index.js`, `dist/jsx-runtime.js`, `dist/jsx-dev-runtime.js` built and valid.
</objective>
