Add these tags after the existing <link rel="icon"> tag:

```html

<meta property="og:title" content="Streem Docs — Getting Started" />

<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Streem Docs — Getting Started" />
<meta name="twitter:description" content="Learn Streem — a TypeScript framework with fine-grained signals and real-time streaming primitives. Guides, API reference, and examples." />
<meta name="twitter:image" content="https://{owner}.github.io/streem-2/logo.svg" />
```
