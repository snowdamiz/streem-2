   - Full example: a typed counter with `signal<number>`, `computed<string>`, and `effect`

2. **JSX configuration** — Show the required tsconfig.json fields:
   ```json
   {
     "compilerOptions": {
       "jsx": "react-jsx",
       "jsxImportSource": "streem"
     }
   }
   ```
   Explain: `jsxImportSource: "streem"` tells TypeScript to use Streem's JSX factory (`streem/jsx-runtime`). This enables type checking for all intrinsic elements and the `class` prop (not `className`). Note: component functions return `Node`, not `JSX.Element`.

3. **CSSProperties** — Show the import and usage patterns:
   ```typescript
   import type { CSSProperties } from 'streem'
   const gridStyle: CSSProperties = { display: 'grid', gap: '16px' }
   // Static: <div style={gridStyle} />
   // Reactive: <div style={() => ({ opacity: isVisible() ? 1 : 0 })} />
   ```
   Note: The style prop accepts `CSSProperties` (static) or `() => CSSProperties` (reactive accessor).

4. **Conditional classes** — Show the idiomatic pattern since Streem has no `clsx` built-in:
   ```typescript
   // Template literal (simple):
   class={`${styles.root} ${isActive ? styles.active : ''}`}
   // Array filter (multiple conditions):
   class={[styles.root, isActive && styles.active, isDisabled && styles.disabled].filter(Boolean).join(' ')}
   // Reactive conditional class (re-evaluates when signal changes):
   class={() => `${styles.root} ${isActive() ? styles.active : ''}`}
   ```

5. **Component prop types** — Show the full prop typing pattern:
   ```typescript
   interface CardProps {
     title: string
     subtitle?: string
     children?: unknown   // use unknown, not ReactNode
     onClick?: () => void
   }
   function Card(props: CardProps): Node {
     return (
       <div class={styles.card} on:click={props.onClick}>
         <h2>{props.title}</h2>
         {props.subtitle && <p>{props.subtitle}</p>}
         {props.children as Node}
       </div>
     ) as unknown as Node
   }
   ```
   Note: `children` typed as `unknown` because Streem children can be `Node | Node[] | string | Signal<T> | (() => T)` — casting to `Node` at the call site is intentional and safe.
