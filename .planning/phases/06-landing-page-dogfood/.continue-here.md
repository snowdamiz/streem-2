- bindStyle fix: handle string style values with el.style.cssText instead of Object.assign (which iterates string chars as indexed properties)
- History cycle fix: replaced row.history() read inside main stream effect with plain row.historyBuf array to avoid subscribing the effect to a signal it also writes to
- SVG sparkline fix: h.ts uses document.createElement (wrong namespace for SVG). Workaround: create SVG via wrapper.innerHTML, extract SVGElement, attach reactive effect manually via path.setAttribute('d', ...)
- Show vs Suspense: Suspense retries re-call children(), creating new streams each time → infinite skeleton. Used Show+hasData signal instead (stream created once in TickerDemo scope).
